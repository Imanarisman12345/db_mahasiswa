# Deskripsi Aplikasi
Aplikasi ini adalah sistem manajemen data mahasiswa berbasis website menggunakan PHP dan MySQL.


## Penggunaan Login

- **Password:** admin123  

---

## Contoh Program
### Halaman Login
Akses di: [Halaman Login](http://localhost/db_mahasiswa/login.php)

### Dashboard
Menampilkan data mahasiswa lengkap dengan fitur tambah, edit, hapus, dan ekspor ke Excel.

---

## Cara Menggunakan
1. Pastikan **XAMPP** telah aktif (Apache dan MySQL).
2. Letakkan folder proyek di `C:\xampp\htdocs\db_mahasiswa`.
3. Buat database bernama `db_mahasiswa` di phpMyAdmin.
4. Import file SQL ke database `db_mahasiswa`.
5. Akses aplikasi di browser melalui:  
6. Login menggunakan username dan password di atas.

---

## Laporan
- Semua data mahasiswa dapat diekspor ke Excel.
- Admin dapat mengelola data mahasiswa melalui dashboard.
